How to access it on macOS

  1. Save this index.html file in ~/Documents/books/quantum.
  2. Confirm that penrose-dirac-holo.mp3 is in that same folder.
  3. Double-click index.html in Finder, then select the audio download link above.
  4. If your browser blocks a local download, open Terminal and run:
    cd ~/Documents/books/quantum && python3 -m http.server 8000
    Then open http://localhost:8000 in your browser.
